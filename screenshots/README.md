## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Optimize for web (under 500KB per image)
- **Resolution**: 1920x1080 for desktop, 375x812 for mobile
- **Naming**: Use kebab-case (e.g., `course-detail-page.png`)
